## How to start working with an external system or integration ?

1. Read the docs
2. See wheather the application support OpenId
3. Hit this into the browser : your-application-baseUrl/.well-known/openid_configuration
4. See the well known properties or endpoints
5. Go for Auth 2.0 Connection
6. Create everything in postman.
7. Create about 5/7 Integration which you want to do!
8. Send the request and save into your postman
9. Go to Salesforce & Create the custom meta data for the integration
10. Start working with the applicaiton.
